## 📌 ДАР БОРАИ ЛОИҲА
Ин лоиҳа як вебсайти муосир ва мутобиқшаванда (Responsive) барои ширкати таҳияи барномаҳо мебошад, ки аз рӯйи дизайни Figma сохта шудааст. Дар сомона маълумот дар бораи ширкат, хизматрасониҳо, таҷриба, фикру мулоҳизаҳои муштариён ва роҳҳои тамос пешниҳод карда мешавад.
Ҳадафи асосии ин лоиҳа омӯзиши табдил додани дизайн аз Figma ба вебсайти воқеӣ бо истифода аз технологияҳои муосири Front-End мебошад.
